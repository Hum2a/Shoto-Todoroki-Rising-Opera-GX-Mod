# Splash Screen

Add an MP4 video (1080×1080, ~5 seconds, H.264 + AAC) for the mod loading screen.

**Structure:** Already in manifest. Replace `splash/todoroki.mp4` with your video.

**Specs:**
- Resolution: 1080×1080 recommended
- Duration: ~5 seconds
- Format: MP4 (H.264 video, AAC audio)

**Create:** Use wallpaper_generator.html as reference, record with OBS, export as MP4. Or animate ice/fire in After Effects, DaVinci Resolve, etc.
