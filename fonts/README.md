# Fonts

Add TTF or WOFF font files here for the mod.

**Structure:** Place one or more font files (e.g. `todoroki.ttf`) and add to manifest:
```json
"fonts": [
  { "id": "1", "name": "Todoroki", "path": "fonts/todoroki.ttf" }
]
```

**Suggestions:** Use fonts.google.com for free fonts (download TTF). Pick something bold/anime-adjacent to match the MHA vibe.
